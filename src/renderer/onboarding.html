Speech-to-text engine
Choose how your voice gets transcribed. Whisper runs locally on your Mac — no API key needed.
Whisper (base.en) Ready
Local, private, free — no API key. ~142 MB model, downloaded once.
Groq cloud
Fast cloud transcription. Requires a free API key.
macOS speech recognition
Built-in, no setup. Lower accuracy than Whisper.
Free at console.groq.com
Refinement
An LLM cleans up transcriptions — fixing grammar, spelling, and technical terms. Optional, and every option here is free or local.
Claude CLI
Uses your installed claude CLI. Fast and high quality.
Codex CLI
Uses your installed codex CLI.
Ollama local
Run a local LLM. Requires Ollama running with a model pulled.
None
Skip refinement. Raw transcription is inserted as-is.
Project context optional
Point Echo at your codebase so it learns your project's terminology, class names, and jargon. This makes transcription much more accurate for technical speech.
