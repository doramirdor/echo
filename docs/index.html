How it works

Four steps, one keypress.

  1. 1

    Record

    Press ⌘⇧V or hold fn and start talking.

  2. 2

    Transcribe

    Speech becomes text via local Whisper, Groq, or macOS Speech.

  3. 3

    Refine

    An LLM cleans it up with context from your apps and vocabulary.

  4. 4

    Insert

    The final text is typed at your cursor — right where you were.

Private by design

Your voice stays on your Mac.

Run Echo fully offline: local Whisper for transcription and no LLM (or a local one). In that mode it makes zero network calls — nothing to log in to, nothing to leak.

  • On-device transcription with whisper.cpp
  • Optional local LLM refinement (Ollama / Llama.cpp)
  • No account, no telemetry, no subscription
  • Cloud providers are strictly opt-in, documented per-provider

For developers

Dictate code and jargon — accurately.

Point Echo at a codebase and it learns your class names, function names, and project vocabulary. Technical terms get biased into the transcriber before the LLM ever runs, so run_pipeline doesn't become "run pipe line."

  • Codebase scanning for project terminology
  • Custom vocabulary + auto-learned corrections
  • Caret-aware sentence continuation
  • Voice commands & reusable templates
